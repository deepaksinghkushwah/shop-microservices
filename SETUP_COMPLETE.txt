🚀 QUICK START

1. Run Tests:
   ./test.sh

2. Build All Services:
   ./build.sh

3. Start Services:
   make dist/start
   # Or manually:
   cd dist && ./start_all.sh

4. Check Services Are Running:
   make dist/status

5. Access APIs:
   Auth:    http://localhost:8081/swagger
   Catalog: http://localhost:8082/swagger
   Order:   http://localhost:8083/swagger

6. Stop Services:
   make dist/stop
   # Or: cd dist && ./stop_all.sh
